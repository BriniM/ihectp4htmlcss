Les feuilles de styles CSS

Historique Syntaxe types de styles Liaison

Le langage HTML a été créé avant tout pour décrire le contenu de la page, alors que les feuilles de style permettent de décrire le style d'affichage de la page. L'utilisation des feuilles de style permet un gain de temps dans la création des sites web et permettent d'éviter l'encombrement du code HTML par les balises de mise en forme.

Historique

La balise <font> qui permet la mise en forme du contenu html a été introduite dans html 3.2. Mais cette balise a créée un probléme d'encombrement des pages web. En plus, pour changer le contenu des pages du site, il faut modifier les balise <font> de cette page ce qui nécessite beaucoup de temps de travail. La solution est arrivée avec le html 4.01 qui permet de lier les pages web avec des feuilles de style qui sont des fichiers écrit avec le langage css créé par le w3c.

Syntaxe

Sélecteur balise Sélecteur identifiant Sélecteur class
balise {déclaration;} #identifiant {déclaration;} .classe {déclaration;}
p {text-indent:30px;} #header {background-color:gray;} .parag {text-align:right;}

Type de style

  • Feuille de style externe : fichier css
  • Feuille de style interne : balise <style>
  • Style dans une balise

Liaison avec une feuille de style

Pour lier un fichier html avec une feuille de style, il faut utiliser la balise <link>
Exemple : La feuille de style est le fichier style.css qui est enregistré dans le méme chemin que la page html:
<link href="style.css" type="text/css" rel="stylesheet">
